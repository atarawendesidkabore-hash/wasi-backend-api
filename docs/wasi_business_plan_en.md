# 1. EXECUTIVE SUMMARY

## The Company

WASI (West African Shipping & Economic Intelligence) is a data infrastructure company that operates the quality assurance, aggregation, and commercialization layer for sovereign government data in West Africa.

WASI does not own the data. WASI does not collect the data directly. **WASI is the trusted technical operator** that governments contract to:

1. **Validate** — ensure data from citizens and businesses is accurate, consistent, and fraud-free
2. **Aggregate** — anonymize and structure raw data into commercially valuable datasets
3. **Commercialize** — sell verified data to global buyers (AI companies, fintech, commodity traders, NGOs, development banks) via API
4. **Distribute payments** — manage the mobile money disbursement pipeline to millions of citizen data contributors
5. **Monitor public contracts** — operate the milestone verification and community labor payment platform
6. **Route cross-border payments** — operate the eCFA CBDC ledger and WASI-Pay interoperability layer for cross-border trade settlement across 15 ECOWAS countries

## The Opportunity

West Africa's 400+ million people generate billions of economic data points daily — none of which are captured in structured, machine-readable form. Simultaneously, the global AI economy has created a multi-billion dollar market for training data, with Africa representing the largest gap.

WASI bridges this gap by partnering with governments who hold the sovereign right to commercialize their national data, and providing the technical infrastructure and quality assurance to make that data sellable at premium prices.

## 4.2 Product 1: Data Validation Engine

**What it does**: Ensures every data point submitted by citizens and businesses is accurate before it enters the commercial dataset.

**How it works**:

| Guardrail | Function | Example |
|-----------|----------|---------|
| G1 — Data Quality Gate | Reject low-confidence data | Reject if confidence score < 0.30, warn if < 0.60 |
| G2 — Feature Attribution | Weight sub-component contributions | Shipping 40%, Trade 30%, Infrastructure 20%, Economic 10% |
| G3 — Calibration (Platt Scaling) | Normalize scores to prevent extreme outliers | k=8.0, pulls toward 50 at low confidence |
| G4 — Human-in-the-Loop | Flag anomalies for manual review | Auto-flag on bank credit, low confidence, >20pt MoM change |

**Cross-validation matrix**:
```
Citizen reports maize = 200 CFA/kg in Kaya
  ↕ cross-check
Business reports maize wholesale = 180 CFA/kg in Kaya region
  ↕ cross-check
WFP historical data shows maize range = 170-220 CFA/kg this season
  ↕ cross-check
Neighboring market (Ouahigouya) reports maize = 195 CFA/kg

Agreement: 4/4 sources consistent → VALIDATED → confidence: 0.92
```

**Key differentiator**: No other data provider in West Africa has multi-source cross-validation. This is what makes WASI data premium-priced.

## 7.4 Village Data Chief Network

The Village Data Chief (VDC) is the critical field-level role:

**Recruitment**: Partner with existing NGO community networks (Plan International, OXFAM, Catholic Relief Services) who already have village-level presence.

**Training**: 5-day program covering USSD platform usage, data quality standards, basic smartphone operation, conflict resolution, fraud detection.

**Compensation**: 40,000 - 60,000 CFA/month (10% of data sales from their village + flat stipend during ramp-up).

**Responsibilities**:
- Recruit and onboard 20+ citizen reporters in their village
- Validate data quality (cross-check reported prices with market visits)
- Resolve disputes and technical issues
- Serve as liaison between WASI and the village
- Report weekly to Provincial Coordinator

**Scale**: ~8,000 villages in Burkina Faso → 8,000 VDC positions at full national coverage.

## 7.5 Quality Assurance Process

```
RAW DATA IN                     VALIDATED DATA OUT
     │                                ▲
     ▼                                │
┌─────────┐                    ┌──────┴──────┐
│ Citizen  │                    │ Commercial  │
│ submits  │                    │ API         │
│ via USSD │                    │ (buyers)    │
└────┬─────┘                    └──────▲──────┘
     │                                │
     ▼                                │
┌─────────────────────────────────────┤
│         VALIDATION PIPELINE         │
│                                     │
│  Step 1: Format check               │
│    Is the data structurally valid?  │
│    → Reject malformed entries       │
│                                     │
│  Step 2: Range check                │
│    Is maize at 200 CFA/kg          │
│    within historical range?         │
│    → Flag if >2 standard devs      │
│                                     │
│  Step 3: Cross-validation           │
│    Do other reporters in the same   │
│    area report similar values?      │
│    → Flag if <3 corroborating      │
│      sources                        │
│                                     │
│  Step 4: Temporal consistency       │
│    Did this reporter report         │
│    200 CFA/kg yesterday and         │
│    800 CFA/kg today?                │
│    → Flag if >50% daily change     │
│                                     │
│  Step 5: Confidence scoring         │
│    Assign 0.00-1.00 confidence      │
│    Green (>0.80) / Yellow (0.60+)  │
│    Red (0.30+) / Grey (<0.30)      │
│    → Reject Grey, warn on Red      │
│                                     │
│  Step 6: Human review (G4)          │
│    VDC or analyst reviews           │
│    flagged entries                   │
│    → Approve, correct, or reject   │
│                                     │
└─────────────────────────────────────┘
```

## 9.1 Moat Analysis

|-----------|-------------|---------------|
| **Government exclusivity** | Concession contracts with 5-year terms + non-compete | Legal barrier — competitor cannot enter without government breach |
| **Ground network** | 8,000+ Village Data Chiefs + 200,000+ citizen reporters | Takes 2+ years to replicate; network effects make each new reporter more valuable |

| **AI-native distribution (MCP)** | Built for autonomous AI agent consumption | Architectural advantage — competitors would need to rebuild |
| **WASI Composite Index** | Proprietary formula with 2+ years of historical data | Data asset — historical data cannot be recreated retroactively |
| **USSD + mobile money integration** | Works on any phone, pays instantly | Operational complexity that deters new entrants |

# 10. RISK ANALYSIS & MITIGATION

| Risk | Probability | Impact | Mitigation |
|------|------------|--------|-----------|
| **Government change or instability** | Medium | High | Structure as multi-ministry PPP; embed in Finance Law (harder to revoke than a decree); diversify across 3+ countries |
| **Low initial data buyer demand** | Medium | Medium | Free tier drives adoption; direct outreach to WFP/World Bank who have immediate need; AI company partnerships |
| **Data quality issues** | Medium | High | 4-level ML guardrails (already built); cross-validation; VDC network for human verification |

| **Mobile money provider issues** | Low | Medium | Multi-provider integration (Orange + Wave + MTN); pre-funded float accounts |
| **Competitor enters market** | Low | Medium | Government exclusivity clause; 2-year head start; network effects |
| **Currency risk (CFA/USD)** | Low | Low | CFA is pegged to EUR; data sales priced in USD; local costs in CFA — natural hedge |
| **Privacy/data breach** | Low | High | SHA-256 anonymization; aggregation thresholds; annual CIL audit; TLS 1.3 encryption |
| **Slow government bureaucracy** | High | Medium | Start with executive decree (faster than legislation); pilot proves value before full legal framework |
| **Terrorism disrupts field operations** | Medium | Medium | USSD works remotely (no field presence needed in hotspots); VDCs are local residents, not outsiders |

## Appendix A: WASI Index Methodology

The WASI Composite Index measures economic health across 16 ECOWAS countries using four sub-components:

| Sub-Component | Weight | Data Sources |
|--------------|--------|-------------|
| Shipping & Port Activity | 40% | Port clearance data, USSD reports, vessel tracking |
| Trade Flows | 30% | UN Comtrade, bilateral trade, USSD border declarations |
| Infrastructure | 20% | Transport composite, road conditions, USSD reports |
| Economic Indicators | 10% | IMF WEO, World Bank, commodity prices |

Each country score is weighted by its economic significance to the region:
- Primary (75%): NG 28%, CI 22%, GH 15%, SN 10%
- Secondary (20%): BF 4%, ML 4%, GN 4%, BJ 3%, TG 3%
- Tertiary (5%): NE, MR, GW, SL, LR, GM, CV — 1% each
